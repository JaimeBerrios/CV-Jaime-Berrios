# Plantillas

Correspondencia entre currículums y clases de LaTeX:

| Currículum | Plantilla |
|---|---|
| `Mark_I_CV_Jaime_Berrios.tex` | `Mark_I_Plantilla.cls` |
| `Mark_II_CV_Jaime_Berrios.tex` | `Mark_II_Plantilla.cls` |
| `Mark_III_CV_Jaime_Berrios.tex` | `Mark_III_Plantilla.cls` |
| `Mark_IV_CV_Jaime_Berrios.tex` | `Mark_IV_Plantilla.cls` |
| `Mark_V_CV_Jaime_Berrios.tex` | No utiliza clase propia; usa `article` y TikZ |
| `Mark_VI_CV_Jaime_Berrios.tex` | `Mark_VI_Plantilla.cls` |
| `Mark_VII_CV_Jaime_Berrios.tex` | No utiliza clase propia; usa `article` y TikZ |

Las plantillas son independientes. Una modificación en una clase afecta
únicamente al Mark que lleva el mismo número.
